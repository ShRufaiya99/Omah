# 🏠 Omah - Modern Real Estate Landing Page

This project is a high-fidelity frontend implementation of the **Omah** real estate landing page. It showcases advanced layout techniques for a clean and professional user interface.

### 🚀 Key Technical Features:
- **Advanced Layouts:** Combining **CSS Grid** for main structures and **Flexbox** for internal alignments.
- **Precise Positioning:** Using `absolute` and `relative` positioning for creative overlapping elements.
- **Responsive Design:** Optimized for a seamless experience across all screen sizes.
- **Typography & Icons:** Custom **font-styles** and icon integration for a premium aesthetic.
- **Media Optimization:** Strategic use of high-quality **images** and custom **color palettes**.

### 🛠️ Technologies Used:
- **HTML5:** For semantic and well-structured markup.
- **CSS3:** Focused on Grid, Flexbox, Positioning, and Media Queries.
